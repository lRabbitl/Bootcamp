Introducción a HTML

¿Qué es?

HTML (HyperText Markup Language) es un lenguaje de marcado de hipertexto que se utiliza para estructurar y presentar el contenido de una página web.
El HTML es la base de todas las páginas web y se utiliza para definir la estructura y el significado del contenido que se muestra en un navegador web. Para ello, utiliza etiquetas y elementos que permiten incluir texto, imágenes, enlaces, videos y otros recursos multimedia.
